## Try This
Before running any AI-generated shell command that touches the filesystem, add a mandatory dry-run step to your prompt: *"Before providing the final command, show me exactly what string each shell layer (zsh, PowerShell, cmd, etc.) will receive after parsing, then provide a `--dry-run` or `-WhatIf` version I should run first."* This forces the model to surface escape-collapse bugs like the Windows wipe incident before any bytes are touched — treat it as non-negotiable for any destructive operation.
