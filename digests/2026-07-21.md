## Tool Updates

- [Claude Code v2.1.216](https://github.com/anthropics/claude-code/releases/tag/v2.1.216) — Critical fix: message normalization cost was growing *quadratically* with session length, causing multi-second stalls and slow resumes in long sessions. Also fixed: OAuth mid-session token expiry causing 401 errors, background agent sessions reverting to default agent on resume (losing your prompt and tool restrictions), `AskUserQuestion` ignoring "wait" answers, and worktree isolation bugs redirecting git operations into shared checkouts. Major release — update immediately if you run long or multi-agent sessions.

- [Codex alpha releases (v0.145.0-alpha.25 through .28)](https://github.com/openai/codex/releases/tag/rust-v0.145.0-alpha.28) — Four alpha bumps with no release notes; active iteration but nothing specific to evaluate.

## Try This

The Claude Code v2.1.216 fix for background agent sessions reverting to default is worth validating immediately: start a background agent session with a custom system prompt and specific tool restrictions, let it run, then resume it and verify the prompt and tool config are actually restored. Before this fix, resumed sessions silently dropped your customizations — meaning any specialised agent persona or restricted toolset you thought was active wasn't. Run a quick sanity check with `claude attach <session-id>` and inspect the active configuration before trusting resumed agent output.
