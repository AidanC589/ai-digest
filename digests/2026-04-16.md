## Tool Updates
- [Claude Code v2.1.110](https://github.com/anthropics/claude-code/releases/tag/v2.1.110) — Large release. Key additions: `/tui fullscreen` for flicker-free rendering, Claude can now send mobile push notifications, `autoScrollEnabled` config, and `--resume`/`--continue` now resurrects scheduled tasks. **Critical security fixes**: command injection hardened against untrusted filenames in "Open in editor," and a permissions bug fixed where `PermissionRequest` hooks returning `updatedInput` weren't being re-checked against `permissions.deny` rules — update immediately if you use permission hooks.
- [Codex 0.121.0](https://github.com/openai/codex/releases/tag/rust-v0.121.0) — Adds a plugin marketplace (`codex marketplace add`) supporting GitHub, git URLs, and local installs; Ctrl+R reverse search in TUI prompt history; expanded MCP support with parallel-call opt-in; and a secure devcontainer profile with bubblewrap sandboxing. Supply chain hardening via pinned GitHub Actions and cargo-deny allowlists also landed.
- [Elevated errors on Claude.ai, API, Claude Code – April 15](https://claudestatus.com/) — Claude.ai and Claude Code login were fully down on April 15; the API recovered by 8:01 AM PT but login remained broken longer. Claude Code users already logged in could continue working. Worth knowing if you hit authentication failures yesterday.
