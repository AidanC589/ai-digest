Tool Updates

Claude Code v2.1.160v2.1.160
Security-focused release: Claude Code now prompts before writing to shell startup files (.zshenv, .zlogin, .bash_login) and build-tool configs that grant code execution (.npmrc, .yarnrc*, .bazelrc, .pre-commit-config.yaml, .devcontainer/). The dynamic-workflow trigger keyword is renamed from workflow to ultracode (violet-highlighted in input). Also fixed: WSL clipboard via PowerShell interop, background session history loss on resume, CJK IME positioning, and Windows input freezing under heavy CPU load.
Codex v0.136.0v0.136.0
Adds session archiving (/archive command), clickable web links in TUI markdown, Windows sandbox alpha provisioning, and remote execution via CODEX_API_KEY. Security hardening prevents /diff from running repo-provided Git hooks and rejects browser-origin exec-server WebSocket handshakes. Bedrock auth now falls back to AWS_REGION/AWS_DEFAULT_REGION.
