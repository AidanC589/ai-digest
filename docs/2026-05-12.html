Engineering & AI Workflows

If AI writes your code, why use Python?
The argument is concrete: Rust's tight compiler feedback loop lets AI agents self-correct in real time using error messages as training signals, making it easier for AI to write correct Rust than correct C++. Real evidence includes a 100K-line C compiler in Rust (~$20K in Claude Code sessions) and a 25K-line JS engine port completed in two weeks.
Quoting James Shore on AI maintenance costs
A sharp warning: if AI doubles your code output but doesn't halve maintenance costs, you've permanently doubled your burden. The implication for reviewers — don't just approve AI code for correctness, actively evaluate whether it's *simpler and easier to maintain* than what you'd have written manually.
Natural-language messages between LLM agents are an architectural anti-pattern
Proposes the "Clipboard Pattern": route structured TypedDict state between agents instead of free-text messages to eliminate lossy re-encoding, semantic drift, and untestable string I/O. Directly actionable if you're designing multi-agent pipelines with LangGraph or similar frameworks.
Shopify's River coding agent operates entirely in public Slack channels
Tobias Lütke describes making all agent sessions public and searchable so the whole org learns by osmosis — every prompt, review, and correction visible to anyone. A concrete org-design pattern for teams wanting to spread AI workflow literacy without formal training programs.
I let AI build a tool to help me figure out what was waking me up at night
A practical account of the "AI expands the threshold of buildable projects" effect: a full Home Assistant + Raspberry Pi + sleep-data correlation tool completed over a weekend. Useful framing for scoping what's now worth attempting with AI assistance vs. dismissing as too complex.

Tool Updates

Claude Code v2.1.139v2.1.139
Major release. Highlights: Agent View (Research Preview) gives a single dashboard of all running/blocked/done sessions — run claude agents to start dispatching multiple parallel sessions without terminal tabs. New /goal command lets you set a completion condition and Claude keeps working across turns until it's met, with a live overlay showing elapsed time, turns, and tokens. Also added: /scroll-speed tuning, claude plugin details <name> with per-session token cost projection, hook args: string[] for shell-safe execution, continueOnBlock for PostToolUse hooks to feed rejection reasons back to Claude, and CLAUDE_PROJECT_DIR in MCP stdio server environments. Notable fixes include a credential deadlock, unbounded SSE memory growth, and broken wildcard Skill permissions.
OpenAI Codex rust-v0.131.0-alpha.6 through alpha.9v0.131.0
Four alpha releases in quick succession; no changelogs published beyond version bumps, suggesting active internal iteration.
