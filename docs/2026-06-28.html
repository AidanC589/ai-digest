Even Google still believes in small models for coding
Google is running hackathons around Gemma 4 31B at 1,500 tokens/second for AI-assisted software engineering, signalling that fast, small, locally-runnable models remain a serious workflow component — not just a fallback for those without API access.
Wayfinder Router: deterministic routing of queries between local and hosted LLM
A new open-source router that deterministically decides whether a query goes to a local or hosted LLM. Relevant for cost control and latency optimisation in multi-step agentic workflows where not every call needs a frontier model.
DSpark: Speculative decoding accelerates LLM inference
DeepSeek's DSpark paper (751 HN points, 315 comments) details speculative decoding techniques that materially speed up inference. Worth understanding if you're optimising local model setups or evaluating hosted inference providers on latency.
Running GLM5.2 on budget hardware < $2500
A practical parts list for running frontier-class models locally under $2,500 using P40s and 512GB RAM. Slow for agents, but viable for planning and debugging passes — and fully export-control-proof.
