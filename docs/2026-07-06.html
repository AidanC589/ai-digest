Engineering & AI Workflows

sqlite-utils 4.0rc3
Simon Willison describes using Claude Fable 5 and GPT-5.5 in parallel to work through a large backlog of issues and PRs simultaneously, shipping compound foreign key support and case-insensitive column handling. Running two frontier models side-by-side on the same codebase is a concrete pattern worth borrowing for non-trivial feature work.
alirezarezvani/claude-skills
A 337-skill library for Claude Code covering 30+ agents, 70+ custom commands, and domain-specific skills (compliance, finance, C-level advisory). If you're not already maintaining a skills/commands library for Claude Code, this is a ready-made starting point.
sponsors/asgeirtj — Extracted system prompts
Leaked/extracted system prompts from Claude Fable 5, Opus 4.8, Claude Code, GPT-5.5, Codex, Gemini 3.5, Cursor, and more in one repo. Reading competitor system prompts is one of the fastest ways to find prompt patterns you're not using.
sponsors/JuliusBrussee — Caveman token-cutting Claude Code skill
A Claude Code skill claiming 65% token reduction by instructing the model to respond in compressed, low-token language. 85K stars suggests widespread adoption; worth benchmarking against your current context costs.
sponsors/Leonxlnx — Taste-Skill
A prompt/skill layer designed to stop AI from generating "boring, generic slop" by giving it aesthetic taste constraints. Addresses one of the most common complaints about AI-generated code and copy quality.

Tool Updates

openai/codex-plugin-cc
Official OpenAI plugin enabling Codex to be called from within Claude Code for code review or task delegation. 25K+ stars and 1,500+ stars added today — this is the most significant tooling bridge between Anthropic and OpenAI workflows yet; worth integrating into your review pipeline immediately.
sqlite-utils 4.0rc3
Release candidate adds compound foreign key introspection/creation and SQLite-standard case-insensitive column name handling; stable 4.0 imminent. Relevant if you use sqlite-utils in any AI-generated data pipeline work.
