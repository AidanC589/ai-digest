Engineering & AI Workflows

Prompt Injection as Role Confusion
Research shows LLMs classify text by *style* rather than *position*, meaning an attacker who mimics the format of a <system> or <think> block can override safety training with 61% success — dropping to just 10% if the injected text is "destyled." Direct implication: AI-generated code that processes untrusted text cannot be assumed safe without explicit destyling defences.
Red-Teaming after Mythos — Zico Kolter & Matt Fredrikson, Gray Swan
OpenAI board member and CMU researchers explain why prompt injection in agentic systems like Claude Code and Codex is a categorically different threat class from traditional security bugs; specialized red-teaming models now outperform humans at breaking AI systems, and the "lethal trifecta" (untrusted data + private data + exfiltration path) is increasingly present in production agent setups.
Why is NO one talking about Microsoft's open source Fast Context!!!
FastContext is a 4B-parameter subagent that handles repo exploration separately from task execution, improving SWE-bench Pro accuracy by ~5 points while cutting tokens by up to 60%; the compact 4B-RL version outperforms the 30B-SFT version, making this practical to run locally alongside your main coding agent.
Porting the Moebius 0.2B image inpainting model to run in the browser with Claude Code
Simon Willison's detailed workflow shows a productive pattern: use Claude.ai for feasibility research and save the output as research.md, then hand that context to Claude Code to begin implementation — effectively using one AI pass to front-load the second. Also demonstrates running Claude Code and Codex Desktop in parallel on different tasks to eliminate idle waiting time.
Investigating Linguistic Steering: An Analysis of Adjectival Effects Across Large Language Model Architectures
Quantitative study across o3, GPT-4o-mini, Llama-3-70B, and DeepSeek-R1 finds that a small subset of adjectives have outsized and non-universal steering effects on model output; larger models show non-additive adjective interactions that are harder to predict, undermining the idea that a prompt that works on one model will transfer reliably to another.

Before handing a new, unfamiliar codebase to Claude Code, first ask Claude.ai (or regular Claude in your IDE) to clone or read the repo and produce a research.md covering: the architectural patterns in use, the riskiest or most complex files, and any security surface areas worth watching. Save that file into the project root, then open Claude Code with the instruction: "Read research.md before touching anything." This front-loads context the agent actually needs, reduces exploratory tool calls, and gives you a human-readable audit trail of what the agent was told — directly addressing the token-waste problem that FastContext is designed to solve.
