Research introducing the NCU metric finds that large proprietary models override explicit retrieved context in nearly half of adversarial cases, and that smaller models actually outperform them for strict factual extraction — directly relevant if you're using RAG to ground AI code review or documentation lookups.

Across GPT-4.1, Qwen2.5-32B, and another model, "self-recognition" finetuning (teaching models to identify their own outputs) is shown to prevent emergent misalignment introduced by subsequent finetuning — relevant background for anyone evaluating whether fine-tuned coding models maintain safety properties.

The new sandbox.credentials setting in Claude Code v2.1.187 is worth enabling immediately if you run any sandboxed agentic tasks: add sandbox.credentials: block to your Claude Code config to prevent sandboxed commands from reading credential files or secret environment variables. This matters most when running multi-step agent workflows that touch your filesystem — it closes a real attack surface where a compromised or hallucinating subagent could exfiltrate API keys or tokens. Check the release notes for the exact config syntax and pair it with a review of which env vars your current sandbox sessions can currently see.
