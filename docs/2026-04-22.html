Meta to start capturing employee mouse movements, keystrokes for AI training data
Meta is installing tracking software called MCI on employee computers to record mouse movements, clicks, keystrokes, and screen snapshots to train autonomous work agents; CTO Andrew Bosworth framed the goal as agents doing "the work" while humans direct and review — a candid preview of where enterprise AI workflows are heading, alongside a 10% workforce cut.

Engineering & AI Workflows

Is Claude Code going to cost $100/month? Probably not — it's all very confusing
Anthropic silently updated pricing to remove Claude Code from the $20 Pro plan, then reversed within hours. The episode is worth reading for Simon Willison's strategic framing: if you're building workflows or teaching others around Claude Code, the pricing instability is a real risk to account for — and an official response came via a tweet, not a changelog.
Anthropic response to Claude Code change
Anthropic's official explanation confirms that agentic usage has fundamentally changed what a "subscription" means: long-running async agents, Claude Code, and Cowork have driven per-subscriber compute far beyond what plans were designed for. Expect further restructuring — existing subscribers get advance notice, but the direction is clearly toward higher pricing tiers for heavy agentic use.
An open letter to Anthropic
A Max-tier user's detailed account of Claude 4.7 introducing hallucinated data, unprompted work orders, and protocol drift across large multi-project pipelines before being caught by a 4.6 audit. Relevant flag for anyone running AI-assisted workflows at scale: model upgrades can silently degrade output quality in ways that only surface through systematic auditing, not spot checks.
Changes to GitHub Copilot Individual plans
Copilot has paused new individual plan signups, restricted Claude Opus 4.7 to the $39/month Pro+ tier, dropped older Opus models, and switched to token-based weekly usage limits. The root cause is identical to Anthropic's problem: agentic sessions burn orders of magnitude more tokens than the original per-request pricing assumed.
Quoting Andreas Påhlsson-Notini
A concise diagnosis worth keeping: "Faced with an awkward task, they drift towards the familiar. Faced with hard constraints, they start negotiating with reality." This describes the review problem precisely — AI agents fail gracefully on easy tasks and fail badly on edge cases, which is exactly when you need them most.

Tool Updates

Claude Code v2.1.117v2.1.117
Large release. Key fixes: Opus 4.7 sessions were computing context against a 200K window instead of the native 1M, causing premature autocompact — now fixed. Default effort for Pro/Max on Opus 4.6 and Sonnet 4.6 upgraded from medium to high. Forked subagents now available on external builds via CLAUDE_CODE_FORK_SUBAGENT=1. macOS/Linux native builds replace Glob and Grep tools with embedded bfs and ugrep for faster searches. OAuth token refresh now happens reactively on 401s instead of dying mid-session. The /resume command now summarises stale large sessions before re-reading.
OpenAI Codex 0.123.0-alpha.5/6/7
Three rapid alpha releases with no detailed changelogs published; active iteration continues on the Rust rewrite.
