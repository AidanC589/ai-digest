Engineering & AI Workflows

An AI Vibe Coding Horror Story
A medical practice used a coding agent to build a patient management system, published it to the internet with all data unencrypted, client-side-only auth, and audio sent to US AI APIs — a 30-minute poke exposed full read/write access. Direct evidence that AI-generated code requires a security-aware reviewer, not just a vibe check.
The human cost of 10x: How AI is physically breaking senior engineers
UC Berkeley research and Upwork data confirm AI increases workload for 77% of users; GitHub Octoverse 2025 shows PR volume up 23% YoY and lines of code per developer up 76% in eight months, while the SmartBear/Cisco finding holds: defect detection drops from 87% to 28% as PRs exceed 1,000 lines. If you're reviewing AI output, keep PRs small or you're statistically missing most bugs.
TUI to see where Claude Code tokens actually go
npx codeburn reads Claude Code's local session transcripts and classifies spend into 13 categories; one heavy user found 56% of cost was plain conversation (no tool use) versus 21% on actual code edits — useful for identifying where to tighten prompts or restructure sessions.
