The AI revolution in math has arrived
Mathematicians including Terence Tao report that 2025–2026 marked a genuine inflection point: AI (including Claude and Gemini) is now helping produce journal-quality proofs and discover new results, with some conjecture-to-proof pipelines running with minimal human intervention. Relevant for anyone thinking about AI's ceiling on reasoning-heavy tasks.

Engineering & AI Workflows

An AI Vibe Coding Horror Story
A medical practice built a patient management system with an AI coding agent, published it with zero backend access control (all auth logic in client-side JS), unencrypted data on a US server without a DPA, and voice recordings sent to AI APIs without patient consent. A direct case study in why AI-generated code requires a reviewer who understands what was built — not just that it runs.
The human cost of 10x: How AI is physically breaking senior engineers
UC Berkeley research embedded in a 200-person company found AI doesn't reduce work, it intensifies it via task expansion, blurred work boundaries, and implicit pressure. Key stat: 88% burnout rate among the highest AI-productivity workers, who are also twice as likely to quit. PR review quality drops from 87% defect detection under 100 lines to 28% over 1,000 lines — and AI is driving PR volume up 98%.
TUI to see where Claude Code tokens actually go
A developer spending $200+/day on Claude Code built codeburn (npx codeburn), which parses session transcripts from ~/.claude/projects/ and classifies spend by task type without any LLM calls. Finding: 56% of spend was pure conversation turns with no tool use; actual file edits were only 21%. Useful for identifying where your prompts are inefficient.
Steve Yegge (via Simon Willison)
