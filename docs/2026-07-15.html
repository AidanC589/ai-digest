Claude Code v2.1.210v2.1.210
Major release with a security-critical fix: the Agent tool has been hardened against indirect prompt injection via content a subagent read — this is directly relevant to anyone running unattended or multi-agent sessions. Also fixed: isolation: 'worktree' subagents could previously run git-mutating commands against the main repo (now fixed); hook callback timeouts were being misreported as user rejections, causing unattended sessions to stall; and ultracode keyword was incorrectly firing on webhook payloads and PR comments. New: live elapsed-time counter on long-running tool calls, startup warnings for deprecated Write(path) permission rules (use Edit(path) or Read(path)).
