Both labs switched enterprise plans to API-rate pricing as of April 2026, meaning companies running coding agents at scale are now paying full token costs; Simon Willison estimates his own 30-day Claude Code usage would have cost ~$1,200 at API rates vs. $100 on the Max plan — a meaningful signal for anyone budgeting agentic workflows at enterprise scale.

Engineering & AI Workflows

sqlite AGENTS.md
SQLite has published a formal AI policy explicitly rejecting agentic code contributions, and has spun off a dedicated bug forum after being flooded with AI-generated reports of varying quality — a practical reminder that auto-submitting agent-generated issues or PRs to open source projects is causing real upstream friction.
Anthropic just confirmed why 90% of non-coding AI agents fail in production
Anthropic's own usage data shows ~50% of agentic activity is software engineering; the thread makes a well-evidenced case that agents fail outside coding not due to model limits but because of dirty data inputs — the practical fix being markdown-converted inputs and guardrail layers before irreversible actions.
Tech CEOs are apparently suffering from AI psychosis
Box's Aaron Levie articulates something directly relevant to AI-assisted devs: executives underestimate the last-mile review work (catching hallucinated libraries, buggy logic, insecure code) because they only see the prototype stage — useful framing when pushing back on unrealistic automation expectations.
Investigating how prompt politeness affects LLM accuracy (2025)
Peer-reviewed study on whether phrasing politeness in prompts measurably affects output quality; worth knowing the empirical answer before building prompt templates at scale.
Claude Is Starting to Feel "Tired", Trying to Avoid Work
Multiple Opus 4.7 + Claude Code users reporting the model spontaneously inserting "should we stop here?" options and questioning explicitly stated task steps — a behavioral regression worth monitoring in long agentic sessions, potentially warranting explicit system prompt instructions to complete tasks without soliciting early exits.
