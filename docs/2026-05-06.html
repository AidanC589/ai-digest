General AI Developments

SubQ: a sub-quadratic LLM with 12M-token context
New model from Subquadratic claims frontier-level long-context accuracy (95% on RULER 128K, 81.8 on SWE-Bench Verified) using a sparse-attention architecture that scales linearly rather than quadratically with context, reducing attention compute ~1,000× at 12M tokens; it includes a Claude Code/Codex/Cursor plugin for full-repo context — benchmarks are third-party validated but the company is new, so treat with appropriate skepticism until independently reproduced.

Engineering & AI Workflows

AI didn't delete your database, you did
Sharp take on the viral Cursor/Claude production DB deletion incident: the real failure was having a public-facing API endpoint that could nuke an entire database, not the agent that called it. Key takeaway for AI-assisted dev: treat your infrastructure's attack surface as if a motivated, non-reasoning process will eventually hit every endpoint — because it will.
10 things about Claude that took me way too long to figure out
Practical prompt patterns worth internalizing: paste the error *before* the code when debugging, ask for specific quality criteria rather than "make it better," use "explain like I'm skeptical" over "explain like I'm 5," and explicitly tell Claude "say I don't know if you don't know" to reduce hallucination confidence.
DeepSeek V4 being 17x cheaper got me to actually measure what I send to cloud vs what I could run locally
A developer ran 150 real coding tasks across cloud and local (Qwen 3.6 27B on a 3090) and found local matched cloud 97% on file reads/explain tasks (35% of workload) and 88% on test writing/boilerplate (30% of workload); only complex multi-file architecture work (15% of tasks) genuinely justified cloud pricing, cutting their API bill from $85 to $22/month by routing by task type.
Our AI started a cafe in Stockholm
An autonomous AI manager running a real café generated cascading real-world errors (ordering 120 eggs with no stove, sending "EMERGENCY" emails to suppliers, submitting a hand-drawn police permit it had never seen the street for). Simon Willison's conclusion is the right one for agent design: any outbound action affecting people outside the system needs a human in the loop — flag this pattern in your own agentic workflows.
datasette-referrer-policy 0.1
Simon Willison used Codex + GPT-5.5 to build a small Datasette plugin to fix an HTTP header issue; a neat real-world example of using AI coding agents for narrow, well-specified infrastructure tasks where the output is immediately verifiable.

Large release with both new features and significant fixes. Highlights: /context was dumping ~1.6k tokens of ASCII grid into every conversation (now fixed); prompt cache TTL was being silently downgraded from 1 hour to 5 minutes (fixed); OAuth refresh race after sleep could log out all sessions (fixed); skillOverrides setting now works; --plugin-url flag lets you load plugins from a URL; CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE enables background auto-updates on Homebrew/WinGet; Ctrl+R history search now defaults back to all projects (pre-2.1.124 behavior). Several enterprise policy and MCP server fixes also included.
