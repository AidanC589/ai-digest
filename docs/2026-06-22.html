Ships as a native Claude Code plugin, Codex plugin, or AGENTS.md file; spins up 5–10 parallel agents to research a topic and compile it into a persistent, cross-referenced Markdown wiki. Relevant for anyone using Claude Code on large, unfamiliar codebases where building institutional knowledge matters across sessions.
Temporary Cloudflare Accounts for AI agents
npx wrangler deploy --temporary now deploys a Cloudflare Workers project for 60 minutes with no account required, useful for AI agent workflows that need ephemeral live endpoints for testing — Simon Willison validated this using GPT-5.5 xhigh in Codex Desktop.
Good results fine tuning a local LLM like Qwen 3:0.6B to categorize questions
Concrete case study: a 0.6B model went from ~10% to reliable accuracy on classification after fine-tuning with ~850 examples using Unsloth/QLora. Practical evidence that tiny local models can be specialized for pre-processing steps in AI pipelines (e.g., routing, intent classification) rather than relying on larger, costlier models for every task.
Local LLM Inference Optimization: The Complete Guide
A year of llama.cpp experiments distilled into a practical guide covering VRAM fitting, KV cache, MoE layer placement, MTP, and OOM traps. Worth feeding directly to your AI assistant when configuring local model infrastructure.
