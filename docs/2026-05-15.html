Access to frontier AI will soon be limited by economic and security constraints
Anthropic's Mythos cybersecurity model and OpenAI's Daybreak initiative both launched with deliberately restricted partner lists, suggesting that the most capable models will increasingly be gated by national security logic rather than willingness to pay — relevant to any team planning infrastructure around frontier model access.
Ontario auditors find doctors' AI note takers routinely blow basic facts
60% of evaluated AI scribe systems mixed up prescribed drugs in patient notes; a concrete, audited hallucination failure rate that underscores why domain-specific evals and human review remain non-negotiable for high-stakes AI output — directly analogous to risks in AI-generated code without review gates.

Coding agents are making technology choices less permanent, which changes how you should think about stack decisions in AI-assisted projects.

Tool Updates

Claude Code v2.1.142v2.1.142
Fast mode now defaults to Opus 4.7 (set CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1 to pin to 4.6). New claude agents flags include --add-dir, --settings, --mcp-config, --model, --effort, and --permission-mode for configuring background sessions. Critical bug fixes: background sessions now survive macOS sleep/wake (daemon detects clock jumps), fixed crash-loop on binary upgrades (e.g. brew upgrade), fixed Windows deadlock on network drives, and fixed MCP_TOOL_TIMEOUT not actually raising the fetch timeout beyond 60 seconds. Also improved reactive compaction to avoid wasted near-full-context retries.
Codex rust-v0.131.0-alpha.16 through alpha.19v0.131.0
Four alpha releases in quick succession; no detailed changelogs published, indicating active but pre-stable iteration on the Rust rewrite.
