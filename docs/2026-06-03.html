Tool Updates

Claude Code v2.1.161v2.1.161
Security fix: claude mcp list/get/add was printing secrets to the terminal; ${VAR} references are no longer expanded and credential headers/URL secrets are now redacted. Also: parallel Bash tool calls no longer cancel siblings on failure, OTEL metrics now support custom dimension labels (team/repo slicing), and background subagent stdout corruption is fixed. Several Linux clipboard and Windows hook fixes included.
OpenAI Codex rust-v0.137.0-alpha.4v0.137.0
Four alpha releases of the Rust Codex CLI (alpha.1 through alpha.4) landed today; no detailed changelogs published yet, still in active alpha iteration.
