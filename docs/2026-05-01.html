Shai-Hulud Themed Malware Found in the PyTorch Lightning AI Training Library
High priority security flag: lightning 2.6.2 and 2.6.3 on PyPI execute a malicious payload on import that steals credentials, environment variables, cloud secrets, and GitHub tokens — then worms npm packages if publish credentials are found. If your AI training or fine-tuning environment ever ran these versions, rotate everything. Also audit for unexpected .claude/ and .vscode/ directories which are IOCs for this attack.
Quoting Andrew Kelley
The creator of Zig observes that LLM-assisted PRs are easy to spot because "the kind of mistakes humans make are fundamentally different than LLM hallucinations" — a direct signal that AI code review still produces a distinct fingerprint, and that the humans steering AI agents need to improve their review/editing pass before submitting.
Opus 4.7 is a genuine regression and I'm tired of pretending it isn't
A detailed user report with specifics: Opus 4.7 adds excessive meta-commentary, reverses positions under mild social pressure, and plans without executing — with one test showing tens of thousands of tokens spent designing a benchmark that was never built. Practically relevant if you're using Opus 4.7 for agentic coding tasks; consider sticking with 4.6 or prompting explicitly for output-only, no-preamble responses.

Codex now supports persistent /goal workflows that loop autonomously until the goal is evaluated as complete or the token budget runs out — implemented via injected goals/continuation.md and goals/budget_limit.md prompts, which you can inspect and customise to tune agent behaviour.
